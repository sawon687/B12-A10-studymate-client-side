## About StudyMate
StudyMate is a web platform built with the MERN stack that helps students connect, collaborate, and improve their learning outcomes. Users can find study partners based on subjects, study preferences, and locations, making learning interactive, engaging, and goal-oriented.

## Features
- **User Authentication:** Secure signup and login with Firebase Authentication.  
- **Profile Management:** Users can update their profiles including profile picture, subjects, and study mode preferences.  
- **Connect with Peers:** Send and receive partner requests to find study partners easily.  
- **Real-Time Updates:** View partner requests and updates instantly without refreshing the page.  
- **Responsive Design:** Works perfectly on desktop, tablet, and mobile devices.  

## Technologies Used
- **Frontend:** React.js, Tailwind CSS, Framer Motion,axios ,swipe silider,daisyui,sweetalert2,react-router,react-spinners,react icon;
- **Backend:** Node.js, Express.js , 
- **Database:** MongoDB  
- **Authentication:** Firebase Authentication
